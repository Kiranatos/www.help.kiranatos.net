Простейшие элементы языка javascript:

window.document.write("<p>Вывод <strong>строки</strong>на экран</p> <br/>");

Тег (аналог \n в java) <br /> устанавливает перевод строки в том месте, где этот тег находится. В отличие от тега абзаца <p>, использование тега <br /> не добавляет пустой отступ перед строкой. Если текст, в котором используется перевод строки, обтекает плавающий элемент, то с помощью атрибута clear тега <br> можно сделать так, чтобы следующая строка начиналась ниже элемента.

Регулярные выражения:

Регулярные выражения обозначаются символом "/", пример: regV = /регулярное выражение/; Спецсимволы . ( в рег.выражениях используются только для подшаблонов, поэтому их нужно экранировать \. \(

var n = str.search(regV) - поиск этого рег.выражения в строке, возвращает номер позиции (-1, 0-9)

если совпадений не найдено метод search возвращает -1

Примеры для str="сумма 45000 грн"; str.search(regV);

1. var regV=/\d\d\d\d\d/ - поиск 5-ти подряд идущих символов от 0-9

2. var regV=/\d{5}/ - поиск 5-ти подряд идущих символов от 0-9 (сокращённая запись - см. таблица внизу - кванторы)

3. var regV=/^\d{5}/ - поиск 5-ти подряд идущих символов от 0-9, которые должны идти вначале (символ ^) строки!

4. var regV=/^\d{5}$/ - поиск 5-ти подряд идущих символов от 0-9, которые должны идти вначале строки, и эти числа также должны быть последними в строке (символ $) - т.е. после них нечего быть не должно!

5. var regV=/\d\d/g - поиск 2-х подряд идущих символов от 0-9, модификатор шаблона g(глобльный поиск) находит все выражения в строке.

var n = str.match(regV) - поиск рег.выражения в строке, возвращает само выражение

6. Выражение для поиска сайтов

s? - буква s может быть в названии протокола http, а может и не быть

(https?: \ / \ / )? - протокол http(s) может быть, а может и не быть в названии

(www\.)? - протокол www может быть, а может и не быть в названии

[a-z0-9-\.] - допустимые символы в названии: "a-z", "0-9", "-" тире, "." точка

{2,1000} - в названии должно быть не меньше 2 и не больше 1000 символов

g - все выражения в строке. (записываетв матрицу)

i - символы не зависят от регистра.

var n = str.replace( regV , ' . ' ); - заменяет все комы на точки (regV= / , /g ).

Работа с числами. Объект Math.

Функция isNaN(b) - если не число то возвращает true, Если число то false.

Функция var a = parseInt(b) - преобразовывает строку в виде "36", "42 грн", "8,25" в число. Число должно стоять в начале строки, всё что после цифр 0-9 отбрасывется - (36, 42, 8) , Если преобразование не удалось то возвращает NaN.

Функция var a = parseFloat(b) - преобразовывает строку в виде "36", "42.1 грн", "8.25" в число. Число должно стоять в начале строки, всё что после цифр 0-9 "." отбрасывется - (36, 42.1, 8.25) , Если преобразование не удалось то возвращает NaN.

Функция var a = Number(bool) - преобразовывает в число (может преобразовать: "36", true, false; не может преобразовать: "36 шт")

Функция var a = b.toFixed(4) - позволяет зафиксировать количество цифр после точки (чтобы нули не отбрасывались например 1.5000 )

Функция var a = Math.round(b) - стандартное округление чисел

Функция var a = Math.ceil(b) - округление чисел в большую сторону (80.001 -> 81)

Функция var a = Math.floor(b) - округление чисел в большую сторону (80.999 -> 80)
